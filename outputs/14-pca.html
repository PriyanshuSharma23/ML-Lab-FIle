Principal Component Analysis - PCA

The principal components of a collection of points in a real p-space that are a sequence of p direction vectors, where the i-th vector is the direction of a line that best fits the data while being orthogonal to the first i 1 vectors.

The principal components transformation can be defined as a transformation that maps the data to a new coordinate system such that the greatest variance by some scalar projection of the data comes to lie on the first coordinate (called the first principal component), the second greatest variance on the second coordinate, and so on.

The principal components are orthogonal because they are the eigenvectors of the covariance matrix, which is symmetric. PCA is sensitive to the relative scaling of the original variables.

import numpy as np

np.random.seed(0)

In[]:
# Create a 3D dataset
np.random.seed(0)
data_3d = np.random.randn(100, 3)
In[]:
# Initialize PCA for 2D to 1D
pca_2d_to_1d = PCA(n_components=1)
# Fit and transform the 2D data
data_1d = pca_2d_to_1d.fit_transform(data_2d)
In[]:
# Initialize PCA for 3D to 2D
pca_3d_to_2d = PCA(n_components=2)
# Fit and transform the 3D data
data_2d_from_3d = pca_3d_to_2d.fit_transform(data_3d)
In[]:

plt.title("Original 2D Data")

plt.scatter(data_1d, np.zeros_like(data_1d))
